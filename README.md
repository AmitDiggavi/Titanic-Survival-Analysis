**Titanic Survival Analysis**
Overview

This project explores the factors that influenced passenger survival aboard the Titanic using exploratory data analysis (EDA), data cleaning, feature engineering, and data visualization techniques in Python.

The goal of this analysis is to identify which demographic, socioeconomic, and family-related characteristics were most strongly associated with survival outcomes.

**Research Question**: 

What factors most strongly influenced a passenger's likelihood of survival aboard the Titanic?

**Dataset**

**This project uses the Titanic passenger dataset, which contains information about:**
```
Passenger demographics (age, gender)
Passenger class
Ticket fare
Family relationships
Embarkation location
Survival status
```
**Dataset source: Kaggle Titanic Dataset**:
